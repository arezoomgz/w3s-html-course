Ordered HTML List - The Type Attribute

The type attribute of the ol tag, defines the type of the list item marker. you can see examples of different markers for ordered lists below

The list items will be numbered with uppercase letters

  1. red
  2. green
  3. grey

The list items will be numbered with lowercase letters

  1. red
  2. green
  3. grey

The list items will be numbered with uppercase roman numbers

  1. red
  2. green
  3. grey

list items will be numbered with lowercase roman numbers

  1. red
  2. green
  3. grey

list items will be numbered with numbers

  1. red
  2. green
  3. grey
